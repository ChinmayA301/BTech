# BTech Code Projects

Curated BTech-era code and project material recovered from Google Drive.

The focus of this repo is source code and runnable project material, not academic report dumps. Generated folders, virtual environments, IDE caches, installers, executables, and obvious third-party dependency clones were excluded.

## Projects

- `dsa-project/` - Rube Goldberg Machine / data structures project in C++ with sample input and screenshots.
- `sepm-railway-management/` - Railway management web app source with HTML, CSS, JavaScript, Express/Mongoose metadata, and assets.
- `python-practice/` - Python practice scripts from coursework.
- `app-week-6-networking/` - Client/server socket and networking exercise scripts.
- `notebooks/` - Face-mask detection AI notebook.
- `csharp-projects/` - C#/.NET project source for calculator and ASP.NET starter apps, with build/IDE artifacts removed.

## Notes

- Build artifacts such as `bin/`, `obj/`, `.vs/`, `node_modules/`, and virtual environments are intentionally not tracked.
- Third-party folders copied into old project directories were filtered out where identifiable.
